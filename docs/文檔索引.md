## 🧩 專案組件概述

台灣保險新聞聚合器由以下主要組件構成：

1. **爬蟲系統** - 自動從多個專業來源獲取保險新聞
2. **分析引擎** - 處理並分析新聞內容的相關性與重要性
3. **Web 界面** - 提供友好的用戶交互界面
4. **API 服務** - 提供標準化的數據接口
5. **資料庫** - 存儲與管理所有新聞數據
